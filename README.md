# TurboWorks
Machine Learning Interface for FireWorks workflows

To learn more about FireWorks, see the [official documentation] (https://pythonhosted.org/FireWorks/)

### Software Requirements:
- Python 3.X
- NumPy
- SciPy
- Scikit-Learn
- Scikit-Optimize
- FireWorks

### Examples
 - ABC_task
 - Branin_task
